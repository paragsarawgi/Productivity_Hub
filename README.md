Productivity Hub<br>
A personal productivity web app built with vanilla HTML, CSS, and JavaScript — no frameworks, no libraries.<br>
What it does:<br>
<br>
Task management with a to-do list that persists across sessions using localStorage<br>
Tasks support optional Important and Star tags<br>
Tasks can be marked complete or deleted, with smooth CSS animations on both actions<br>
A live progress bar tracks completed vs total tasks with a motivational message<br>
Planner section is currently under construction<br>
<br>
UI and responsiveness:<br>
The interface uses a dark-toned two-panel layout for the to-do section with a warm peach accent color. Fully responsive across desktop, tablet, and mobile using CSS Flexbox and media queries. Includes a hamburger menu on mobile and a navbar that hides on scroll and reappears on intentional upward scroll.
Built with:<br>
HTML, CSS (custom properties, Flexbox, CSS transitions), JavaScript (DOM manipulation, event delegation, localStorage)<br>
Status:<br>
Ongoing. Core to-do functionality is complete. Planner section is in active development.<br>
